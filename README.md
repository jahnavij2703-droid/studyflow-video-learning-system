StudyFlow – Curated Personalized Video Learning System

A web-based learning platform that helps students find syllabus-related educational videos automatically using OCR and YouTube API integration.

Features

- Upload syllabus image
- OCR-based text extraction
- Curated YouTube video recommendations
- User authentication
- Video playback
- Topic-wise learning
- Admin dashboard

Technologies Used

- React JS
- Spring Boot
- Java
- MySQL
- Tesseract OCR
- YouTube Data API

Project Overview

StudyFlow simplifies the learning process by extracting topics from uploaded syllabus images and generating relevant educational video recommendations automatically. The platform reduces manual searching and helps students access organized learning resources efficiently.
